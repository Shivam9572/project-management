# TaskFlow - Task Management System

A professional, production-ready task management system with separate dashboards for Admins and Members, built with modern technologies.

## 🚀 Project Overview

TaskFlow is a comprehensive task management solution that provides role-based access control, real-time task tracking, and analytics dashboards. It features a powerful admin dashboard for managing projects, team members, and tasks, along with a clean member dashboard focused on productivity.

## ✨ Features

### Admin Dashboard
- **Analytics Dashboard** with charts and graphs
- Complete project management (CRUD operations)
- Team member management
- Task assignment and tracking
- Reports and analytics
- Activity logs
- Search and filter functionality

### Member Dashboard
- Personal task management
- Task status updates
- Project viewing
- Calendar view
- Personal progress tracking

### Common Features
- **Authentication** with JWT
- **Role-based access control**
- **Real-time notifications**
- **Drag-and-drop task board**
- **Responsive design** for all devices
- **Dark/Light mode** support
- **Skeleton loading** states
- **Toast notifications**

## 🛠️ Tech Stack

### Backend
- Node.js with Express.js
- PostgreSQL with Sequelize ORM
- JWT for authentication
- Bcrypt for password hashing
- Helmet for security headers
- Rate limiting

### Frontend
- React 18 with TypeScript
- Tailwind CSS for styling
- React Router v6 for routing
- Chart.js for analytics
- Axios for API calls
- React Hot Toast for notifications

## 📋 Prerequisites

- Node.js (v16 or higher)
- PostgreSQL (v12 or higher)
- npm or yarn package manager
